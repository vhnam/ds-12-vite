# InputField

Labelled input field composed of a label, helper text, and an `Input` control with shared validation styling.

## When to use InputField vs Input

Use **InputField** for standard form fields that need a visible label and helper or error text. The wrapper wires `htmlFor`, `aria-describedby`, and validation state automatically.

Use a bare **Input** only in custom layouts where you manage label association yourself — for example, a search bar embedded in a toolbar or a composite field with external labelling.

## Accessibility

- Label is associated with the input via `htmlFor` / `id`
- Helper text is linked through `aria-describedby`
- Invalid state sets `aria-invalid` on the control
- Disabled fields use the native `disabled` attribute
